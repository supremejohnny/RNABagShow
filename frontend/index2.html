Balanced Academic View

RNABag: 面向精准肿瘤学的转录组基础模型

RNABag 将 RNA 表达谱表示为基因 ID 与表达值的联合 token 序列，通过 Transformer encoder 学习跨活检模态的转录组表征，并在组织、血浆和血小板任务上加载专用 checkpoint 完成分类。

Input feature set 4096 HVGs + CLS + 2 summary tokens
Sequence length 4099
Backbone 8-layer Transformer encoder
Downstream tasks 5 checkpoints

Preprocessing

RNA-seq 数据整理

FPKM 表达矩阵经过 GeneID 到 Symbol 映射、样本转置、HVG 过滤和 log1p 转换，生成 `.npy` 推理输入。

Representation

基因与表达联合嵌入

每个位置拼接 32 维基因 ID embedding 与 4 维表达值 projection，再映射到 128 维模型空间。

Prediction

任务检查点切换

不同活检模态和预测目标通过 checkpoint 与 label map 切换，输出癌症状态、组织来源或肿瘤定位。
