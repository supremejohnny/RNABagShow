    3输入规则 · Input contract

    模型接受以基因为观测维度、样本为特征列的 bulk RNA expression matrix。提交前请确认以下数据约定。

    File encoding
    UTF-8 编码的 tab-separated values（.tsv），不接受逗号分隔矩阵或 Excel 工作簿。
    Quantification
    表达丰度须完成 FPKM normalization；矩阵元素应为有限、非负的连续数值。
    Matrix orientation
    第一列为 GeneID，后续每列对应一个独立样本。RNABag 将执行基因映射、4096 HVG selection 与 log1p transformation。
    Sample modality
    当前分析仅接受 tissue RNA；单一矩阵中不应混合不同生物样本模态。
