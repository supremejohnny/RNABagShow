RNABag 转录组基础模型展示

RNABag 面向不同活检模态的 RNA 表达谱，将 4096 个高变异基因映射到统一的 Transformer 表征空间，用于癌症检测、组织来源识别与血小板肿瘤定位等精准肿瘤学任务。

Model Overview

从 RNA-seq 表达到临床任务预测

当前展示页聚焦 RNABag 的项目本体，不接入在线推理服务。页面中的样本结果来自预置演示数据，用于说明模型输入、推理阶段和输出形式。

Data

表达谱标准化

原始 FPKM 数据经过基因映射、样本转置、组织信息匹配、高变异基因过滤和 log1p 转换，形成推理所需的矩阵输入。

Backbone

Transformer 编码器

模型配置包含 128 维隐空间、8 层编码器和 8 个注意力头，以统一表示不同模态来源的转录组信号。

Heads

任务专用检查点

组织、血浆和血小板任务分别加载对应 checkpoint，输出二分类、36 类组织来源或 5 类肿瘤定位结果。

Inference Pipeline

展示版保留真实工程路径

现阶段页面只做静态演示，但内容按 RNABag 仓库中的数据处理和推理入口组织，后续可以平滑接入 FastAPI 服务。

01

Raw RNA-seq

读取 FPKM、GeneID 到 Symbol 映射和样本组织信息。

02

Feature Processing

转置表达矩阵，生成标签，过滤 4096 个 HVG 并做 log1p 变换。

03

Task Selection

根据 tissue、plasma 或 platelet 任务选择 checkpoint 和标签映射。

04

RNABag Inference

通过 Transformer 编码器输出 logits，并转换为概率或类别名称。
