RNABag · Transcriptome foundation model

一个模型，连接多种转录组任务

RNABag 以 TCGA 与 GTEx 表达谱为基础，通过高变基因选择、in silico augmentation 与共享 RNA 编码器，学习可在 solid biopsy 与 liquid biopsy 之间迁移的通用转录组表征。

此处可继续补充模型的训练样本规模、队列覆盖、外部验证策略与主要 benchmark 结果。当前文案用于展示中等篇幅的项目介绍在该文字区域中的排版容量。

还可以在这里介绍模型的适用边界、研究用途、可解释性方法与临床使用声明；即使文字量继续增加，右侧模型总览图仍保持独立、完整的阅读空间。

统一表征

将 4096 个高变基因的表达与基因 ID 编码为可迁移的 RNA representation。

    3输入规则 · Input contract

    模型接受以基因为观测维度、样本为特征列的 bulk RNA expression matrix。提交前请确认以下数据约定。

    File encoding
    UTF-8 编码的 tab-separated values（.tsv），不接受逗号分隔矩阵或 Excel 工作簿。
    Quantification
    表达丰度须完成 FPKM normalization；矩阵元素应为有限、非负的连续数值。
    Matrix orientation
    第一列为 GeneID，后续每列对应一个独立样本。RNABag 将执行基因映射、4096 HVG selection 与 log1p transformation。
    Sample modality
    当前分析仅接受 tissue RNA；单一矩阵中不应混合不同生物样本模态。
