Summary

模型简介

RNABag 接收 log1p 转换后的 RNA 表达向量，以基因 ID 和表达值构造 token 序列，经 Transformer encoder 得到 CLS 表征，再由任务专用 MLP decoder 输出分类 logits。当前推理仓库覆盖 tissue、plasma、platelet 三类活检模态以及五个下游任务。

  • Input genes: 4096 high variability genes
  • Sequence length: 4099 tokens
  • Backbone: 8 Transformer encoder blocks
  • Prediction vector: CLS token output
  • Implementation: PyTorch + Lightning DataModule
  • Batch size: 8 during inference config

Architecture Parameters

核心配置

Parameter Value Meaning
total_ids 4097 4096 gene IDs plus index 0 for special tokens.
seq_len 4099 4096 gene tokens plus CLS and two total-expression summary tokens.
d_id 32 Gene ID embedding dimension.
d_value 4 Expression value projection dimension.
d_model 128 Transformer hidden dimension after fusion MLP.
d_ffn 512 Feed-forward dimension inside each encoder block.
n_layers 8 Number of Transformer encoder blocks.
n_heads 8 Multi-head self-attention heads per block.
dropout 0.1 Dropout used in MLP and attention blocks.
compress_d_model 128 Configured compressed embedding dimension.
compress_seq_len 1 Configured compressed sequence length.
cpu_workers 1 DataLoader worker count in config.

Input Construction

输入序列与前向传播

Gene IDs np.arange(1, len(expr) + 1), then shuffled with expression values.
CLS token Prepended as gene=0, expr=0.
Summary tokens Appended as two gene=0 tokens carrying raw_sum and input_sum.
Embedding fusion EmbeddingInt(4097, 32) + Linear(1, 4), concatenated to 36 dimensions.
Dimension correction MLP: Linear(36,128) -> Dropout -> ReLU -> LayerNorm -> Linear(128,128).
Encoder block Self-attention, residual, LayerNorm, FFN 128 -> 512 -> 128, residual, LayerNorm.
Decoder input encoder_output[:, 0, :], the CLS representation.
Decoder MLP Linear(128,128) -> Dropout -> ReLU -> LayerNorm -> Linear(128,n_labels).

Preprocessing

数据处理路径

Required files fpkm.tsv, gene mapping TSV, info.txt, tcga_hvg_gene_4096.txt.
Gene mapping Map GeneID to Symbol and remove the original GeneID column.
Matrix orientation Transpose FPKM data so samples become rows and genes become columns.
HVG filtering Keep the 4096 configured high-variability genes, filling missing genes with zero.
Transformation Apply np.log1p to the selected expression matrix and save as `.npy`.
Inference loader The inference dataset currently loads log1p_tissue.npy from the configured data directory.
Academic version. Parameters are summarized from the local RNABag inference code and README.
